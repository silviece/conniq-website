# Conniq website

Dit is een statische onepage website voor Conniq.

## Positionering

De website is aangepast naar:

Conceptontwikkeling voor sociale en maatschappelijke vraagstukken, gericht op praktische, communicatieve en digitale oplossingen.

Niet meer opgenomen:
- civiele vraagstukken
- adviseren en begeleiden
- diepgaand onderzoek
- procesbegeleiding

## Bestanden

- `index.html` - de volledige website
- `style.css` - alle styling
- `script.js` - mobiel menu en kleine animaties
- `assets/logo-conniq.png` - het Conniq-logo
- `assets/favicon.svg` - icoon voor browser-tab

## Contactknop aanpassen

Zoek in `index.html` naar:

```html
<a class="btn btn-primary" href="#" aria-label="Contactlink nog toevoegen">Neem contact op</a>
```

Vervang `#` door bijvoorbeeld een LinkedIn-link, afsprakenlink of formulierlink.
